Code of conduct
===============

This Code of Conduct outlines expectations for participation in the
CHB-Bootloader project, whether in issues, pull requests, discussions,
or other interactions.

Expected Behavior
-----------------

All participants are expected to:

- Be respectful and considerate toward others.

- Use clear and constructive language.

- Report problems politely and provide helpful feedback.

- Respect project maintainersâ€™ decisions.

Unacceptable Behavior
---------------------

Unacceptable behavior includes, but is not limited to:

- Harassment, intimidation, or discrimination of any kind.

- Offensive or derogatory comments or content.

- Personal attacks on contributors or maintainers.

- Posting inappropriate images, links, or content.

Reporting Issues
----------------

If you experience or witness any unacceptable behavior, please:

1. Contact the project maintainers directly by opening an issue or
   sending an email (example: andrescamorao9@gmail.com)

2. Provide details about the incident.

3. Keep communication respectful and factual.

Enforcement
-----------

Project maintainers have the right to take appropriate action to address
unacceptable behavior. This may include:

- Removing comments or content.

- Closing issues or pull requests.

- Blocking contributors from participation if necessary.

Scope
-----

This Code of Conduct applies to all project spaces, including:

- GitHub repository (issues, pull requests, discussions)

- Project-related chats, forums, and emails.

- In-person or virtual meetings related to CHB-Bootloader.
